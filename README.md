# Installation

0. Clone git
1. [Install Python](https://www.python.org/downloads/)
2. [Install Poetry](https://python-poetry.org/docs/#installing-with-the-official-installer)
3. Init project
4. Run app/main.py

# Init project
1. Install poetry
```
poetry install
poetry shell
```
2. Create .env file (example: .env.dev)
3. Run project
```
python app/main.py
```
